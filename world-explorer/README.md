# World Explorer

World Explorer is a Next.js project that allows users to explore countries around the world.

## Features

- App Router
- File-based routing
- Shared layout
- Dynamic routes
- Server components
- Client components
- Real API data fetching
- Static rendering and caching
- Dynamic rendering
- Search functionality
- Dark mode
- Favorites system
- Responsive design

## API Used

REST Countries API

## Run Locally

npm install
npm run dev
